## 프롬프트 유형과 선택 이유
```
선택한 유형: 롤플레잉(Role-Playing) + 체인 오브 스루트(Chain of Thought, CoT)

선택 이유:

롤플레잉 (Role-Playing): 문제 상황에서 '광고 대행사의 콘텐츠 제작 팀장'이라는 명확한 페르소나와 전문성이 요구됩니다. 링크의 '역할 부여' 유형은 "전문성 확보"가 주요 목적이므로, 해당 역할에 몰입하여 전문가적 관점에서 문제를 해결하도록 지시하기에 적합합니다.

체인 오브 스루트 (CoT): 이 문제의 핵심 훈련 초점은 **[원인 분석]**입니다. 링크에서 CoT 유형은 "복잡한 추론"과 "문제 해결"에 적합하며, "최종 답변에 도달하기까지의 추론 과정을 단계별로 설명하도록 유도"합니다. C 영상이 실패한 원인(CTR, 이탈률)을 논리적으로 추론하고, 그 근거를 바탕으로 해결책을 도출하는 데 가장 효과적인 유형입니다.
```

## 작성 위해 추가 필요 요건
```
문제를 해결하고 구체적인 대응 방안을 보고하기 위해 다음 자료가 추가로 필요합니다.

정량적 데이터:

C 영상의 시청자 유지율(Audience Retention) 그래프: 80%가 이탈하는 정확한 3초 구간을 확인하고, 어떤 시각/청각 요소에서 이탈이 발생하는지 분석해야 합니다.

A, B, C 영상의 노출 타겟 오디언스 데이터: C 영상에 가장 많은 예산이 투입되었다면, 혹시 타겟 설정이 잘못되어 관련 없는 사용자에게 노출이 집중되었는지 확인해야 합니다.

A, B 영상의 시청자 유지율 그래프: 성공한 영상과 실패한 영상을 직접 비교하여 차이점을 분석해야 합니다.

정성적 자료:

C 영상 원본 파일: 영상의 첫 3초(이탈이 발생하는 구간)에 할인 혜택이 명확하게 전달되는지, 혹은 너무 자극적이거나 불쾌한 요소가 있는지 직접 확인해야 합니다.

A, B 영상 원본 파일: 성공한 영상의 소구 방식(성분, 후기)과 C 영상의 소구 방식을 비교 분석해야 합니다.

초기 광고 기획안 및 크리에이티브 브리프: C 영상의 핵심 메시지와 타겟 고객이 초기 전략과 일치하는지 검토해야 합니다.
```

### 1. C 영상(할인 혜택) 실패 원인 분석 (Chain of Thought)

#### 추론 1 (3초 이탈률 80%): "3초 안에 80%가 이탈한다"는 데이터의 의미
* **가상 데이터:** C 영상의 시청자 유지율 그래프를 확인한 결과, 80%의 이탈은 0.5초에서 1.5초 사이에 집중적으로 발생했습니다.
* **추론:** 6초 범퍼 광고에서 1.5초 이내의 이탈은 영상의 내용이 아닌, **도입부의 즉각적인 거부감**을 의미합니다. C 영상은 "지금 즉시 50% SALE!"이라는 자극적인 텍스트와 귀에 거슬리는 효과음을 첫 0.5초에 배치했습니다.
* **결론:** 시청자는 이 광고를 '신규 뷰티 브랜드'의 정보가 아닌, '저품질 스팸 광고'로 인식하여 즉각적으로 'Skip' 버튼을 눌렀습니다. 3초까지 기다릴 가치조차 느끼지 못한 것입니다.

#### 추론 2 (소구점의 역설): 왜 '할인'이 '성분'과 '후기'보다 실패했는가
* **추론 (신뢰의 부재):** 클라이언트 'A'는 **신규 브랜드**입니다. 소비자는 이 브랜드가 무엇인지, 신뢰할 수 있는지 모릅니다. A(성분)와 B(후기) 영상은 각각 '제품의 가치'와 '사회적 증거(신뢰)'를 구축하며 브랜드의 신뢰도를 쌓았습니다.
* **추론 (가치의 전제):** '할인' 소구는 소비자가 이미 해당 제품의 '정가(Value)'를 알고 있거나 브랜드를 신뢰할 때 강력하게 작동합니다. 하지만 C 영상은 **가치 증명 없이 가격 할인만**을 제시했습니다. 소비자 입장에선 "들어본 적 없는 제품을 할인한다"는 정보가 매력적이지 않으며, 오히려 "신뢰할 수 없는 제품"이라는 인상을 강화했습니다.
* **결론:** A, B 영상이 브랜드 신뢰를 구축하는 동안, C 영상은 신뢰가 없는 상태에서 가격만 제시하여 실패했습니다. 신규 브랜드에게는 'Why to Buy(구매 이유)'가 'How much(가격)'보다 중요합니다.

#### 추론 3 (A/B 영상과의 비교): 성공과 실패의 결정적 차이
* **가상 데이터:** A/B 영상은 타겟이 '뷰티 성분 관심 유저', '인기 뷰티 유튜버 구독자'로 비교적 정교하게 설정된 반면, C 영상은 예산이 많아 '18-45세 여성, 뷰티 관심사'로 매우 광범위하게 설정되었습니다.
* **추론 (타겟의 질):** A/B는 구매 가능성이 높은 잠재 고객에게 노출되어 높은 반응(CTR 2.5%)을 이끌어냈습니다.
* **추론 (메시지와 타겟의 불일치):** C 영상은 브랜드를 전혀 모르는 불특정 다수에게 노출되었습니다. 이들에게 "할인" 메시지는 효과가 없었으며, 자극적인 도입부(추론 1)와 결합하여 최악의 성과(CTR 0.8%)를 기록했습니다.
* **결론:** A/B 영상은 **'정확한 타겟 + 가치(신뢰) 제안'**으로 성공했습니다. C 영상은 **'불명확한 타겟 + 신뢰 없는 가격 제안'**으로 실패했습니다.

#### 결론: C 영상 핵심 실패 원인 3가지
1.  **공격적이고 저렴한 도입부:** 영상 시작 1.5초 이내의 자극적인 'SALE' 효과음과 디자인이 즉각적인 광고 이탈(80%)을 유발했습니다.
2.  **신규 브랜드의 신뢰도 간과:** 가치 증명(A, B) 없이 가격 할인(C)을 먼저 제시하여, 신뢰가 없는 소비자들의 구매 장벽을 넘지 못했습니다.
3.  **예산 낭비를 유발한 광범위한 타겟팅:** 가장 많은 예산을 사용했음에도, 브랜드 인지도가 없는 불특정 다수에게 광고가 노출되어 CTR과 전환율이 급락했습니다.

### 2. 긴급 대응 및 수정/대체 방안 (Action Plan)

#### 즉각 조치
1.  **C 영상 광고 집행 즉시 중단:** 추가적인 예산 낭비를 막기 위해 C 캠페인을 즉시 Pause 합니다.
2.  **예산 재분배:** C 영상의 잔여 예산을 성과가 검증된 **A 영상(성분)과 B 영상(후기)으로 즉시 이전**하여 캠페인 전체 효율을 극대화합니다.
3.  **A/B 영상 타겟 확장 테스트:** A/B 영상의 타겟을 C 영상의 광범위한 타겟 중 일부와 결합하여 효율을 테스트합니다.

#### A안 (즉시 수정: Re-edit)
기존 C 영상 소스를 활용하되, 실패 원인을 제거하는 방향으로 즉시 재편집합니다.

1.  **[수정안 1] 신뢰 우선, 혜택 후면 (A+C 결합):**
    * **(0-3초):** A 영상(성분)의 성공적인 도입부("독점 특허 성분 90%")를 그대로 사용합니다.
    * **(4-6초):** 제품 컷과 함께 차분한 톤으로 "지금 런칭 혜택으로 경험하세요"라는 카피를 노출합니다. (자극적인 "SALE", "%" 문구 삭제)
2.  **[수정안 2] 후기 우선, 혜택 연결 (B+C 결합):**
    * **(0-3초):** B 영상(후기)의 임팩트 있는 후기 자막("인생템 만났어요")을 먼저 보여줍니다.
    * **(4-6초):** "A 브랜드가 드리는 첫 구매 혜택"이라는 문구로 자연스럽게 혜택을 연결합니다.

#### B안 (신규 대체: Retargeting)
C 영상을 완전히 대체하며, A/B 영상의 성공 요소를 혜택과 결합합니다.

* **[대체안] A/B 영상 시청자 리타겟팅 전용 범퍼:**
    * **타겟:** A 영상 또는 B 영상을 시청했으나 구매하지 않은 유저.
    * **메시지:** 이들은 이미 브랜드의 가치(성분, 후기)를 인지하고 있습니다. 이들에게만 '할인' 혜택을 노출합니다.
    * **카피 예시:** (차분한 톤으로) "고민되셨나요? A 브랜드가 첫 구매를 응원합니다. (15% OFF 쿠폰)"
    * **기대 효과:** 가장 구매 전환 가능성이 높은 타겟에게 혜택을 제시하여 C 영상의 본래 목적인 '전환율'을 극대화합니다.
